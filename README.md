🎮 [기획 업데이트 내용 정리]

1. 기본 화면 및 UI 레이아웃
2D 픽셀 아트 스타일: 주인공(모험가)과 몬스터(무지개 슬라임).
ESC 메뉴: 게임 중 ESC 키 입력 시 즉시 일시 정지(Time.timeScale = 0).
메뉴 구성
좌측 || 게임 재개, 게임 종료(메인 타이틀로 이동)
우측 || 현재 누적된 능력치(스탯) 확인.

3. 조작 및 기본 전투 메커니즘
대시 (생존기): Spacebar를 눌러 바라보는 방향으로 짧고 빠르게 이동. 약 0.5초의 무적 판정을 제공하며 쿨타임이 존재합니다.
타격 연출 (폴리싱): 타격 시 몬스터가 하얗게 깜빡이며, 머리 위로 데미지 텍스트가 떠오릅니다. 폭탄 아이템 사용 시 화면 흔들림(Screen Shake) 연출이 추가.

5. 무기 시스템 및 진화
단검 (근거리): 캐릭터가 바라보는 전방 180도(반원) 범위 내 가장 가까운 적만 공격. 매우 강력하지만 뒤쪽이 취약합니다.
샷건 (중거리): 맵의 50% 거리. 전방 최대 60도(좌우 30도) 범위 내 적을 타격. 고정 데미지.
저격 (원거리): 맵 끝까지 도달. 거리가 멀수록 데미지 증가. 단, 공격 속도는 다른 무기의 30% 수준으로 매우 느리게 조정.
장판/오라 : 테스트 필요

*** 무기 진화 (시너지): 무기가 최대 레벨에 도달하고 특정 스탯(예: 이속, 속성 등)을 갖췄을 때, 보스 상자를 통해 화상/빙결 등의 상태 이상이 추가된 상위 무기로 진화합니다.

4. 몬스터 스폰 및 웨이브 로직
변수 창출: 넉백에 면역을 가지거나 덜 밀려나는 '무장 슬라임'이 아주 낮은 확률로 섞여 등장합니다.

5. 성장 및 스탯 시스템 (인게임)
이동속도 : 이동속도와 gem 수집범위 분리
방어력: 방여력 선택시 증가폭  증가
초당 체력 회복: 1초마다 N만큼의 체력을 자동 회복.
최대 체력 증가: 선택 시 최대 체력이 늘어나며, 증가된 전체 체력의 10%만큼 즉시 회복 (최대 체력 초과 불가).

7. 드롭 아이템 및 필드 기믹
필드 상호작용: 맵의 랜덤한 위치에 생성
현재 아이템은 3분마다 리스폰 - 없어지지 않음

전체 제거 (폭탄): 획득 시 필드 위 모든 몬스터 즉시 사망 (경험치는 정상 드롭).
타임 스탑 (시계): 획득 시 10초간 몬스터 스폰 및 기존 몬스터 이동 완전 정지.
폭발 드럼통  : 타격 시 광역 데미지
랜덤 버프 제단 : 일시적 공속/이속 증가

7. 영구 성장 시스템 (메타 프로그레션)
8. 몬스터가 드롭하는 **'골드'**를 모아 게임 오버 후 메인 메뉴 상점에서 캐릭터의 기본 스탯(초기 체력, 공격력 등)을 영구적으로 강화하거나 신규 캐릭터를 해금합니다.
